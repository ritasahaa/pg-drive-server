# PG Drive Backend

REST API backend for PG Drive Rental Management System.

## Features

- JWT Authentication
- User Management
- Role Based Access
- Booking APIs
- MongoDB Integration
- Error Handling

## Tech Stack

- Node.js
- Express.js
- MongoDB
- JWT
- Mongoose

## API Modules

- Authentication
- Users
- PG Management
- Bike Management
- Booking Management

## Installation

git clone <repository-url>

npm install

npm run dev
